# Turbina wiatrowa

Projekt przedstawia uproszczony system sterowania turbiną wiatrową.  
Program symuluje podstawowe mechanizmy kontroli pracy turbiny, takie jak:

- odczyt danych z czujników,
- ustawianie turbiny względem kierunku wiatru,
- regulacja kąta łopat,
- obsługa sytuacji awaryjnych.

## Cel projektu

Celem projektu jest przedstawienie podstaw działania automatycznego systemu sterowania turbiny wiatrowej oraz symulacja pracy układu w różnych warunkach środowiskowych.

## Funkcje programu

- monitoring parametrów pracy,
- kontrola bezpieczeństwa,
- automatyczne ustawianie gondoli,
- regulacja pracy turbiny,
